# Traceify

Open-source black & white image to SVG vectorizer (client-side).
Upload → adjust threshold → preview mask → export SVG.

## Features
- Client-side only (your image never leaves the browser)
- Auto invert + manual flip
- Presets: Logo (sharp) / Sticker (smooth)
- Mask preview (BW) + SVG preview
- Download SVG

## Run locally
```bash
npm install
npm run dev
